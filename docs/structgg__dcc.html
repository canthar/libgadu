Detailed Description

Połączenie bezpośrednie do wersji Gadu-Gadu 6.x.

Tworzone przez gg_dcc_socket_create(), gg_dcc_get_file(), gg_dcc_send_file() lub gg_dcc_voice_chat(), zwalniane przez gg_dcc_free().
