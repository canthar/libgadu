Detailed Description

Obsługa wiadomości.

Plik zawiera funkcje dotyczące obsługi "klasy" gg_message_t, które w przyszłości zostaną dołączone do API. Obecnie używane są funkcje konwersji między tekstem z atrybutami i HTML.
